## VStack - 縦方向のレイアウト

VStackはSwiftUIでビューを縦方向に並べて配置するためのコンテナです。

<img src="/images/swiftui-basics/3_1.png" alt="VStack - 縦方向のレイアウト" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    var body: some View {
        VStack(spacing: 20) {
            Text("上")
                .font(.title)
                .foregroundColor(.red)
            
            Text("中央")
                .font(.title)
                .foregroundColor(.green)
            
            Text("下")
                .font(.title)
                .foregroundColor(.blue)
        }
        .padding()
    }
}

#Preview {
    ContentView()
}
```

複数のビューを上から下へ順番に積み重ねるレイアウトを作成できます。

```swift
VStack(spacing: 20) {
    Text("上")
        .font(.title)
        .foregroundColor(.red)
    
    Text("中央")
        .font(.title)
        .foregroundColor(.green)
    
    Text("下")
        .font(.title)
        .foregroundColor(.blue)
}
```

`.font(.title)`はSwiftUIで用意されている標準フォントサイズの一つで、タイトルに適した大きさのフォントを適用します。bodyやheadlineなどの他の標準サイズも利用可能です。

VStackの基本的な使い方は、`VStack { }` のように波括弧の中に並べたいビューを記述します。この例では、赤い「上」、緑の「中央」、青い「下」という3つのTextビューを縦に並べています。

`spacing`パラメータは、ビュー同士の間隔を指定します。この例では`spacing: 20`と指定しているので、各Textビューの間に20ポイントのスペースが挿入されます。spacingを指定しない場合は、デフォルトの間隔が適用されます。

ビューはVStack内に記述した順番で上から下へ配置されます。最初のビューが一番上になり、最後のビューが一番下になります。

VStackは画面のレイアウトを整理するのに非常に便利です。例えば、タイトル、コンテンツ、ボタンなどを縦に並べて配置するようなUIを作成する際に使用します。`padding()`をVStackに適用することで、全体に余白を追加できます。

## HStack - 横方向のレイアウト

HStackはSwiftUIでビューを横方向に並べて配置するためのコンテナです。

<img src="/images/swiftui-basics/3_2.png" alt="HStack - 横方向のレイアウト" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    var body: some View {
        HStack(spacing: 15) {
            Text("左")
                .font(.title)
                .foregroundColor(.red)
            
            Text("中央")
                .font(.title)
                .foregroundColor(.green)
            
            Text("右")
                .font(.title)
                .foregroundColor(.blue)
        }
        .padding()
    }
}

#Preview {
    ContentView()
}
```

複数のビューを左から右へ順番に並べて配置するレイアウトを作成できます。

```swift
HStack(spacing: 15) {
    Text("左")
        .font(.title)
        .foregroundColor(.red)
    
    Text("中央")
        .font(.title)
        .foregroundColor(.green)
    
    Text("右")
        .font(.title)
        .foregroundColor(.blue)
}
```

HStackの基本的な使い方は、`HStack { }` のように波括弧の中に並べたいビューを記述します。この例では、赤い「左」、緑の「中央」、青い「右」という3つのTextビューを横に並べています。

`spacing`パラメータは、ビュー同士の間隔を指定します。この例では`spacing: 15`と指定しているので、各Textビューの間に15ポイントのスペースが挿入されます。spacingを指定しない場合は、デフォルトの間隔が適用されます。

ビューはHStack内に記述した順番で左から右へ配置されます。最初のビューが一番左になり、最後のビューが一番右になります。

HStackは画面のレイアウトを整理するのに非常に便利です。例えば、アイコンとテキストを横に並べたり、複数のボタンを横に配置したりするようなUIを作成する際に使用します。`padding()`をHStackに適用することで、全体に余白を追加できます。

## ZStack - 重ね合わせレイアウト

ZStackはSwiftUIでビューを奥行き方向に重ねて配置するためのコンテナです。

<img src="/images/swiftui-basics/3_3.png" alt="ZStack - 重ね合わせレイアウト" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    var body: some View {
        ZStack {
            Color.blue.opacity(0.3)
                .frame(width: 200, height: 200)
            
            Color.red.opacity(0.5)
                .frame(width: 150, height: 150)
            
            Text("重ねる")
                .foregroundColor(.white)
                .font(.largeTitle)
        }
    }
}

#Preview {
    ContentView()
}
```

複数のビューを同じ位置に重ねて配置するレイアウトを作成できます。

```swift
ZStack {
    Color.blue.opacity(0.3)
        .frame(width: 200, height: 200)
    
    Color.red.opacity(0.5)
        .frame(width: 150, height: 150)
    
    Text("重ねる")
        .foregroundColor(.white)
        .font(.largeTitle)
}
```

ZStackの基本的な使い方は、`ZStack { }` のように波括弧の中に重ねたいビューを記述します。この例では、青い背景、赤い背景、白いテキストの順に重ねています。

ビューはZStack内に記述した順番で奥から手前に重ねられます。最初のビューが一番奥になり、最後のビューが一番手前に表示されます。この例では、青いColorが一番奥、赤いColorが中央、白いTextが一番手前になります。

`opacity()`はビューの透明度を指定します。0.0が完全に透明、1.0が完全に不透明です。この例では青を0.3（30%透明）、赤を0.5（50%透明）に設定しています。

`frame()`はビューのサイズを指定します。widthとheightで幅と高さをポイント単位で設定できます。

ZStackは背景の上に他の要素を重ねるのに適しています。例えば、背景画像の上にテキストを配置したり、バッジのような重ね合わせUIを作成したりする際に使用します。奥行きのあるレイアウトを作成することで、よりリッチな視覚効果を実現できます。

## Spacing - 要素間の間隔

spacingパラメータはVStackやHStack内でビュー同士の間隔を指定するための重要な設定です。

<img src="/images/swiftui-basics/3_4.png" alt="Spacing - 要素間の間隔" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    var body: some View {
        VStack(spacing: 20) {
            
            VStack(spacing: 0) {
                Text("間隔0")
                Text("ぎゅっと")
                Text("くっつく")
            }
            .background(Color.yellow.opacity(0.3))
            
            VStack(spacing: 10) {
                Text("間隔10")
                Text("少し")
                Text("離れる")
            }
            .background(Color.green.opacity(0.3))
            
            VStack(spacing: 30) {
                Text("間隔30")
                Text("かなり")
                Text("離れる")
            }
            .background(Color.blue.opacity(0.3))
        }
        .padding()
    }
}

#Preview {
    ContentView()
}
```

適切な間隔を設定することで、読みやすく使いやすいレイアウトを作成できます。

```swift
VStack(spacing: 20) {
    Text("間隔なし")
        .font(.headline)
    
    VStack(spacing: 0) {
        Text("間隔0")
        Text("ぎゅっと")
        Text("くっつく")
    }
    .background(Color.yellow.opacity(0.3))
    
    VStack(spacing: 10) {
        Text("間隔10")
        Text("少し")
        Text("離れる")
    }
    .background(Color.green.opacity(0.3))
    
    VStack(spacing: 30) {
        Text("間隔30")
        Text("かなり")
        Text("離れる")
    }
    .background(Color.blue.opacity(0.3))
}
```

`spacing`パラメータは`VStack(spacing: 値)` または `HStack(spacing: 値)` のように指定します。値はポイント単位で、0以上の数値を設定できます。

この例では、異なるspacing値を持つ3つのVStackを比較しています：

- **spacing: 0** : ビュー同士が完全に密着します。ぎゅっとくっついたレイアウトになります。

- **spacing: 10** : ビュー同士が少し離れます。適度な間隔で読みやすくなります。

- **spacing: 30** : ビュー同士がかなり離れます。ゆったりとしたレイアウトになります。

`background()`は各VStackに半透明の背景色を付けて、間隔の違いを視覚的にわかりやすくしています。`opacity(0.3)`で30%の透明度を設定しています。

spacingの値はアプリのデザインやコンテンツの種類によって調整します。ボタン同士の場合は少し間隔を空け、関連するテキスト同士の場合は密着させるなど、状況に応じて使い分けます。適切なspacingはUIの使いやすさと美しさを大きく向上させます。
