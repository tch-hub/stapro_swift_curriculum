## Listの基本

アプリ開発では、たくさんのデータを一覧で表示することが頻繁にあります。SwiftUIでこれを実現するのが`List`（リスト）です。

一番シンプルな使い方は、表示したい項目をそのまま並べる方法です。

```swift title="ContentView.swift"
import SwiftUI

struct ContentView: View {
    var body: some View {
        List {
            Text("りんご")
            Text("みかん")
            Text("ぶどう")
        }
    }
}

#Preview {
    ContentView()
}
```

これで、iOSの標準的な設定アプリのような、区切り線の入ったきれいなリストが自動的に作成されます。

## 配列データからリストを作る (ForEach)

実際のアプリでは、項目を直接書くのではなく、配列（データのまとまり）からリストを作ることがほとんどです。
配列の要素を繰り返し表示するには、`List`に配列を直接渡すか、`ForEach`を使います。

まずは、Listに直接配列を渡す方法です。

```swift title="ContentView.swift"
import SwiftUI

struct ContentView: View {
    // 表示したいデータの配列
    let fruits = ["りんご", "みかん", "ぶどう", "いちご", "メロン"]
    
    var body: some View {
        // id: \.self は「データ自身を識別子にする」という意味
        List(fruits, id: \.self) { fruit in
            Text(fruit)
        }
    }
}

#Preview {
    ContentView()
}
```

このコードでは、`fruits`配列の中身を一つずつ取り出し、`fruit`という変数に入れて`Text`で表示しています。
`id: \.self` は、SwiftUIが各項目を区別するための「目印」として、文字列そのものを使うという指定です。

## ListとForEachを組み合わせる

静的な項目と動的な項目（配列データ）を混ぜて使いたい場合や、リストの項目を削除・並び替えしたい場合は、`List`の中に`ForEach`を配置します。

```swift title="ContentView.swift"
import SwiftUI

struct ContentView: View {
    let fruits = ["りんご", "みかん", "ぶどう"]
    
    var body: some View {
        List {
            Section("おすすめ") {
                Text("今日の特売品")
                    .foregroundColor(.red)
            }
            
            Section("フルーツ一覧") {
                ForEach(fruits, id: \.self) { fruit in
                    Text(fruit)
                }
            }
        }
    }
}

#Preview {
    ContentView()
}
```

この例では、`Section`を使ってリストをグループ分けしています。
`List`は非常に強力なビューで、このように少しのコードで情報を整理して見やすく表示することができます。
