## Circle - 円

Circle()で円を描画できます。fill()で塗りつぶし、stroke()で枠線を指定できます。

<img src="/images/swiftui-basics/5_1.png" alt="Circle - 円" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    var body: some View {
        VStack(spacing: 30) {
            // 青い塗りつぶしの円
            Circle()
                .fill(Color.blue)
                .frame(width: 100, height: 100)
            
            // 赤い枠線の円
            Circle()
                .stroke(Color.red, lineWidth: 5)
                .frame(width: 80, height: 80)
            
            // 緑の塗りつぶしに黒い枠線の円
            Circle()
                .fill(Color.green)
                .overlay(
                    Circle()
                        .stroke(Color.black, lineWidth: 4)
                )
                .frame(width: 120, height: 120)
        }
    }
}

#Preview {
    ContentView()
}
```

SwiftUIの図形ビューでは、`fill()`と`stroke()`を使って図形の塗りつぶしと枠線を指定できます。これらのメソッドは図形の見た目を大きく変えます。

## fill() - 塗りつぶし

`fill()`メソッドは図形の内部を指定した色で塗りつぶします。引数にはColorやGradientなどの塗りつぶしスタイルを指定できます。

```swift
.fill(Color.blue)        // 単色で塗りつぶし
.fill(Color.red.opacity(0.5))  // 半透明の色
.fill(LinearGradient(...))     // グラデーション
```

## stroke() - 枠線

`stroke()`メソッドは図形の輪郭に線を描きます。引数には色と線の太さを指定できます。

```swift
.stroke(Color.red, lineWidth: 5)  // 赤い5ポイントの枠線
.stroke(Color.blue, lineWidth: 2)  // 青い2ポイントの枠線
```

## overlay()を使った枠線付き図形

`fill()`と`stroke()`を組み合わせる場合、`overlay()`を使って枠線を重ねることができます。

```swift
Circle()
    .fill(Color.green)           // まず緑で塗りつぶし
    .overlay(
        Circle()                    // 同じサイズの円を重ねて
            .stroke(Color.white, lineWidth: 4)  // 白い枠線を描く
    )
```

`overlay()`を使うと、塗りつぶしの上に別のビューを重ねることができます。これにより、塗りつぶしと枠線の両方を表現できます。

**青い塗りつぶしの円**
```swift
Circle()
    .fill(Color.blue)
    .frame(width: 100, height: 100)
```

**赤い枠線の円**
```swift
Circle()
    .stroke(Color.red, lineWidth: 5)
    .frame(width: 80, height: 80)
```

**緑の塗りつぶしに黒い枠線の円**
```swift
Circle()
    .fill(Color.green)
    .overlay(
        Circle()
            .stroke(Color.black, lineWidth: 4)
    )
    .frame(width: 120, height: 120)
```

## Rectangle - 四角形

Rectangle()で四角形を描画できます。cornerRadius()で角の丸みを調整できます。

<img src="/images/swiftui-basics/5_2.png" alt="Rectangle - 四角形" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    var body: some View {
        VStack(spacing: 20) {
            // 角丸なしの四角形
            Rectangle()
                .fill(Color.orange)
                .frame(width: 150, height: 80)
            
            // 少し角丸の四角形
            Rectangle()
                .fill(Color.blue)
                .frame(width: 150, height: 80)
                .cornerRadius(10)
            
            // かなり角丸の四角形
            Rectangle()
                .fill(Color.green)
                .frame(width: 150, height: 80)
                .cornerRadius(25)
            
            // RoundedRectangleで角丸四角形
            RoundedRectangle(cornerRadius: 20)
                .fill(Color.pink)
                .frame(width: 150, height: 80)
            
            // Capsule（完全な角丸）
            RoundedRectangle(cornerRadius: 40)
                .fill(Color.purple)
                .frame(width: 150, height: 80)
        }
    }
}

#Preview {
    ContentView()
}
```

Rectangle()は四角形を描画するビューです。fill()で塗りつぶし、stroke()で枠線を指定できます。cornerRadius()で角の丸みを調整できます。

## fill()とstroke()の使用例

RectangleでもCircleと同じように`fill()`と`stroke()`を使用できます：

```swift
Rectangle()
    .fill(Color.blue)              // 青い塗りつぶし
    .frame(width: 100, height: 50)

Rectangle()
    .stroke(Color.red, lineWidth: 3)  // 赤い3ポイントの枠線
    .frame(width: 100, height: 50)
```

## cornerRadius() - 角の丸み

Rectangleの角を丸くするには`cornerRadius()`を使用します：

```swift
Rectangle()
    .fill(Color.blue)
    .frame(width: 150, height: 80)
    .cornerRadius(10)  // 10ポイントの角丸
```

数値が大きいほど角が丸くなります。cornerRadiusを指定しない場合は直角の四角形になります。

## RoundedRectangle - 最初から角丸の四角形

RoundedRectangle()を使うと、最初から角丸の四角形を作成できます：

```swift
RoundedRectangle(cornerRadius: 20)
    .fill(Color.pink)
    .frame(width: 150, height: 80)
```

`RoundedRectangle(cornerRadius: 値)`で角丸の半径を指定します。

## Capsule - 完全な角丸（楕円）

角丸の半径を大きくするとCapsule（カプセル）のような形になります：

```swift
RoundedRectangle(cornerRadius: 40)
    .fill(Color.purple)
    .frame(width: 150, height: 80)
```

幅と高さが同じ場合は完全な円になります。

**角丸なしの四角形**
```swift
Rectangle()
    .fill(Color.orange)
    .frame(width: 150, height: 80)
```

**少し角丸の四角形**
```swift
Rectangle()
    .fill(Color.blue)
    .frame(width: 150, height: 80)
    .cornerRadius(10)
```

**RoundedRectangleで角丸四角形**
```swift
RoundedRectangle(cornerRadius: 20)
    .fill(Color.pink)
    .frame(width: 150, height: 80)
```

**Capsule（完全な角丸）**
```swift
RoundedRectangle(cornerRadius: 40)
    .fill(Color.purple)
    .frame(width: 150, height: 80)
```
