## Textビューの基本

TextビューはSwiftUIで最も基本的なビューコンポーネントです。画面にテキストを表示するために使用します。

<img src="/images/swiftui-basics/1_1.png" alt="Textビューの基本" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    var body: some View {
        Text("Hello, World!") // Hello, World!を表示するテキスト
    }
}

#Preview {
    ContentView()
}
```

```swift
Text("Hello, World!")
```
`Text("Hello, World!")`はTextビューを作成しています。Textビューは文字列を引数に取って、その文字列を画面に表示します。この例では`Hello, World!`という文字列が表示されます。
TextビューはSwiftUIの基礎となるビューで、他のビューと組み合わせて複雑なUIを作成する土台となります。シンプルですが、フォント、色、サイズなどの装飾を追加することで様々な表現が可能になります。

## テキストの装飾

Textビューは様々な装飾をチェーンして適用することができます。

<img src="/images/swiftui-basics/1_2.png" alt="テキストの装飾" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    var body: some View {
        Text("SwiftUIの装飾") // 基本のテキストを作成
            .font(.largeTitle) // 大きなタイトル用のフォントサイズ
            .foregroundColor(.blue) // 青色に設定
            .fontWeight(.bold) // 太字に設定
            .multilineTextAlignment(.center) // 複数行の場合中央揃え
            .padding() // 周囲に余白を追加
    }
}

#Preview {
    ContentView() // プレビューを表示
}
```

この例では、基本的なTextビューに複数の装飾を順番に適用しています。

```swift
Text("SwiftUIの装飾")
```
まず、基本のテキストを作成します。この時点では、まだ何の装飾も適用されていません。

```swift
Text("SwiftUIの装飾")
    .font(.largeTitle) //大きなタイトル用のフォントサイズ
    .foregroundColor(.blue) //青色に設定
    .fontWeight(.bold) //太字に設定
```
次に、フォントと色を設定します。

```swift
Text("SwiftUIの装飾")
    .font(.largeTitle)
    .foregroundColor(.blue)
    .fontWeight(.bold)
    .multilineTextAlignment(.center) //複数行の場合中央揃え
    .padding() // 周囲に余白を追加
```
最後に、配置と余白を追加します。

`padding()`は引数なしで呼び出すと、SwiftUIのデフォルトの余白（通常16ポイント）が適用されます。数値を指定することでカスタムな余白を設定することも可能です。例えば、`padding(20)` とすると上下左右に20ポイントの余白が適用されます。また、`padding(.horizontal)` のように特定の方向だけに余白を適用することもできます。

これらの修飾子は、ドット(.)でつなげて順番に適用することができます。SwiftUIではこのようなチェーン構文が一般的で、コードを読みやすく整理できます。

このコードに書かれていない他の装飾パターンとしては、以下のようなものがあります：

```swift
.background(Color.yellow)     // テキストの背景色を設定
.underline()                  // テキストに下線を引く
.strikethrough()              // テキストに取り消し線を引く
.italic()                     // テキストを斜体にする
.baselineOffset(5)            // テキストのベースラインをずらす
.kerning(2)                   // 文字間のスペースを調整
.lineSpacing(5)               // 行間のスペースを調整
.minimumScaleFactor(0.5)      // テキストが収まらない場合の最小縮小率
.lineLimit(2)                 // 表示する最大行数を制限
.textCase(.uppercase)         // テキストを大文字に変換
```

これらの装飾を組み合わせることで、プロフェッショナルで魅力的なテキスト表示を作成できます。装飾の順番によって結果が異なる場合があるので、注意が必要です。

## 複数行テキスト

Textビューは長いテキストを扱う場合、自動的に折り返して複数行で表示します。

<img src="/images/swiftui-basics/1_3.png" alt="複数行のテキスト" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    var body: some View {
        Text("これは長いテキストです。SwiftUIでは自動的に折り返されて複数行で表示されます。\n\n改行を\n明示的に\n入れることも\nできます")
    }
}

#Preview {
    ContentView()
}
```

また、`\n`を使って明示的に改行を入れることもできます。

```swift
Text("これは長いテキストです。SwiftUIでは自動的に折り返されて複数行で表示されます。\n\n改行を\n明示的に\n入れることも\nできます")
```
Text()の中に長い文字列を渡しています。この文字列には通常の文章と`\n`（改行コード）が含まれています。SwiftUIは画面の幅に合わせて自動的にテキストを折り返しますが、`\n`がある場所では必ず改行されます。
このように、Textビューは単純な文字列表示だけでなく、長い文章や改行を含む複雑なテキスト表示にも対応しています。ブログ記事や説明文を表示する際に便利です。
