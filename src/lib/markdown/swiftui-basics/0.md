## 自動生成される基本構造

新しいSwiftUIアプリ制作を作ると、このコードが自動で作られます。これがSwiftUIアプリの基本の形です。

<img src="/images/swiftui-basics/0_1.png" alt="自動生成される基本構造" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI

struct ContentView: View {
    var body: some View {
        Text("Hello, World!")
    }
}

#Preview {
    ContentView()
}
```

```swift
import SwiftUI
```
SwiftUIをよみこむことで、TextやButtonなどの便利な機能を使えるようになります。

```swift
struct ContentView: View {
    var body: some View {
    }
}
```
ContentViewという構造体を作ります。これはアプリの画面のメイン部分です。Viewというルールに従って作ります。bodyという部分は、画面に何を表示するかを書くところです。some Viewは、戻り値の型を指定する特別な書き方です。これでSwiftUIがうまく動きます。

```swift
#Preview {
    ContentView()
}
```

#Previewは、コードを書くたびに画面をすぐに見られる機能です。これで、アプリの見た目をすぐに確認できます。
