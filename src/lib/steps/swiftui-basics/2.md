## Buttonの基本

ButtonはSwiftUIで最も基本的なインタラクティブなビューです。

<img src="/images/swiftui-basics/2_1.png" alt="Buttonの基本" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    @State private var buttonText = "押してみよう"
    
    var body: some View {
        Button(buttonText) {
            buttonText = "押されました！"
        }
    }
}

#Preview {
    ContentView()
}
```

最もシンプルな形では、`Button("ラベル") { action }` という構文で作成できます。

```swift
@State private var buttonText = "押してみよう"
```

`@State`を使ってボタンのテキストを管理します。これにより、ボタンを押すとテキストが変更されます。

```swift
Button(buttonText) {
    buttonText = "押されました！"
}
```

最初の引数にはボタンに表示する文字列（ラベル）を渡します。この例では`buttonText`変数の値がボタンのラベルになります。

2番目の引数にはactionクロージャーを渡します。これはボタンがタップされたときに実行されるコードです。この例では`buttonText`に"押されました！"を代入して、ボタンのテキストを変更します。

ボタンをタップすると、action内のコードが一度だけ実行され、テキストが変更されます。ボタンは自動的にタップ可能な外観を持ち、ユーザーが押したことを視覚的にフィードバックします。

## ボタンのスタイル

Buttonには様々なスタイルがあります。Textビューを使って装飾することで、ボタンの見た目をカスタマイズできます。

<img src="/images/swiftui-basics/2_2.png" alt="ボタンのスタイル" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    @State private var buttonText = "スタイル付きボタン"
    
    var body: some View {
        Button(action: {
            buttonText = "押されました！"
        }) {
            Text(buttonText)
                .padding() // 内側に余白を追加
                .background(Color.green) // 背景色を緑に設定
                .foregroundColor(.white) // 文字色を白に設定
                .cornerRadius(15) // 角を丸くする
                .shadow(radius: 5) // 影を付けて立体感を出す
        }
    }
}

#Preview {
    ContentView()
}
```

Textビューを使って装飾することで、ボタンの見た目をカスタマイズできます。

```swift
@State private var buttonText = "スタイル付きボタン"
```

`@State`を使ってボタンのテキストを管理します。

```swift
Button(action: {
    buttonText = "押されました！"
}) {
    Text(buttonText)
        .padding() // 内側に余白を追加
        .background(Color.green) // 背景色を緑に設定
        .foregroundColor(.white) // 文字色を白に設定
        .cornerRadius(15) // 角を丸くする
        .shadow(radius: 5) // 影を付けて立体感を出す
}
```

- **padding()** : ボタンの内側に余白を追加し、テキストとボタンの境界の間にスペースを作ります。これによりボタンが押しやすくなります。

- **background(Color.green)** : ボタンの背景色を緑色に設定します。Color.greenはSwiftUIで用意されている標準色です。

- **foregroundColor(.white)** : テキストの色を白色に設定します。これにより緑色の背景に白い文字が映えます。

- **cornerRadius(15)** : ボタンの角を15ポイントの半径で丸くします。数値が大きいほど角が丸くなります。

- **shadow(radius: 5)** : ボタンに5ポイントの半径の影を付けて、立体感を出します。影の半径が大きいほど影が柔らかくなります。

ボタンをタップすると、action内のコードが実行され、テキストが"押されました！"に変更されます。これらの修飾子を組み合わせることで、プロフェッショナルな見た目のボタンを作成できます。
