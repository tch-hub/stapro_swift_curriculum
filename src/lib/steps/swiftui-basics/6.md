## TextField - テキスト入力

TextFieldでテキスト入力を扱えます。$記号を使って@State変数とバインディングします。

<img src="/images/swiftui-basics/6_1.png" alt="TextField - テキスト入力" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI

// メインのビュー構造体
struct ContentView: View {
    // ユーザーの名前を保存する状態変数
    // @Stateをつけると値が変わった時に画面が自動更新される
    @State private var name = ""
    
    // 画面に表示する内容を定義するプロパティ
    var body: some View {
        // 縦方向にビューを並べるコンテナ
        VStack(spacing: 20) {
            // 名前を入力するテキストフィールド
            // $nameで状態変数とつなげる（バインディング）
            TextField("名前を入力してください", text: $name)
                .textFieldStyle(RoundedBorderTextFieldStyle()) // 角丸の枠線スタイル
                .padding(.horizontal) // 左右に余白を追加
            
            // 条件分岐：名前が入力されている場合のみ挨拶を表示
            // !name.isEmpty は「nameが空でない」という意味
            if !name.isEmpty {
                // 挨拶メッセージを表示
                Text("こんにちは、\(name)さん！")
                    .font(.title) // 大きなフォントサイズ
                    .foregroundColor(.blue) // 青色で表示
            }
        }
        .padding() // 全体に余白を追加
    }
}

// プレビューでアプリの見た目を確認
#Preview {

}
```

## Toggle - スイッチ

Toggleでオン/オフのスイッチを作成できます。$記号で@State変数とバインディングし、状態に応じてUIを変更できます。

<img src="/images/swiftui-basics/6_2.png" alt="Toggle - スイッチ" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI

// メインのビュー構造体
struct ContentView: View {
    // スイッチのオン/オフ状態を保存する変数
    // Bool型でtrue（オン）またはfalse（オフ）を保存
    @State private var isOn = false
    
    // 画面に表示する内容を定義
    var body: some View {
        // 縦方向にビューを並べるコンテナ
        VStack(spacing: 20) {
            // オン/オフを切り替えるスイッチ
            // $isOnで状態変数とつなげる（バインディング）
            Toggle("通知をオンにする", isOn: $isOn)
                .padding(.horizontal) // 左右に余白を追加
            
            // 三項演算子でスイッチの状態に応じてテキストを変更
            // isOnがtrueなら「通知オン」、falseなら「通知オフ」
            Text(isOn ? "通知オン" : "通知オフ")
                .font(.title) // 大きなフォントサイズ
                .foregroundColor(isOn ? .green : .red) // 状態に応じて色変更
                .padding() // 内側に余白
                .background(
                    // 背景色も状態に応じて変更
                    RoundedRectangle(cornerRadius: 10)
                        .fill(isOn ? Color.green.opacity(0.2) : Color.red.opacity(0.2))
                )
        }
        .padding() // 全体に余白を追加
    }
}

// プレビューでアプリの見た目を確認
#Preview {
    ContentView()
}
```

ToggleはSwiftUIでオン/オフのスイッチを作成するためのビューです。ユーザーが設定を切り替えられるようにします。

## Toggleの基本構文

```swift
Toggle("ラベルテキスト", isOn: $stateVariable)
```
// スイッチのラベルと、@State変数（$記号付き）をバインドします。

- **第1引数**: スイッチの横に表示されるラベル（説明文）
- **isOnパラメータ**: @State変数とのバインディング（$記号必須）

## @Stateとのバインディング

```swift
@State private var isOn = false
```
// スイッチの状態（オン/オフ）を保存する Bool型の変数です。

```swift
Toggle("通知をオンにする", isOn: $isOn)
```
// $isOnで状態変数とバインディングし、スイッチの操作で値が変化します。

ToggleはBool型の@State変数とバインディングします。スイッチがオンになるとisOnがtrueに、オフになるとfalseになります。

## 三項演算子による条件分岐

```swift
Text(isOn ? "通知オン" : "通知オフ")
    .foregroundColor(isOn ? .green : .red)
```
// isOnがtrueなら「通知オン」、falseなら「通知オフ」と表示し、色も変わります。

- `isOn ? A : B`: isOnがtrueならA、falseならB
- これにより、スイッチの状態に応じてテキストや色を即座に変更できます

## 注意点

1. **Bool型**: Toggleは必ずBool型の@State変数とバインディングします
2. **$記号**: バインディングには必ず$記号が必要です
3. **初期値**: false（オフ）またはtrue（オン）から始めます
4. **条件分岐**: 三項演算子を使ってUIを動的に変更できます
5. **ネスト**: 親Toggleの状態に応じて子Toggleを表示/非表示にできます

Toggleは設定画面やオプション選択で欠かせないUIコンポーネントです。

## Slider - スライダー

Sliderで数値をスライドして入力できます。valueパラメータに$記号でバインドし、inパラメータで範囲、stepでステップを指定できます。

<img src="/images/swiftui-basics/6_3.png" alt="Slider - スライダー" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI

// メインのビュー構造体
struct ContentView: View {
    // 音量の値を保存する変数（0.0から1.0の範囲）
    // Float型で小数点以下の値を保存
    @State private var volume = 0.5
    
    // 画面に表示する内容を定義
    var body: some View {
        // 縦方向にビューを並べるコンテナ
        VStack(spacing: 30) {
            // 現在の音量を表示（パーセントに変換）
            // Int(volume * 100)で小数から整数に変換
            Text("音量: \(Int(volume * 100))%")
                .font(.title) // 大きなフォントサイズ
            
            // 音量を調整するスライダー
            // 0...1の範囲で0.1刻み、$volumeでバインディング
            Slider(value: $volume, in: 0...1, step: 0.1)
                .padding(.horizontal) // 左右に余白
                .accentColor(.blue) // 青色のスライダー
            
            // 横方向にアイコンを並べるコンテナ
            HStack {
                // 左側のスピーカーアイコン
                // volume < 0.3（音量が30%未満）の場合のみ青色表示
                Image(systemName: "speaker.wave.1")
                    .foregroundColor(volume < 0.3 ? .gray : .blue)
                
                Spacer() // 間を広げるためのスペース
                
                // 右側のスピーカーアイコン
                // volume > 0.7（音量が70%超）の場合のみ青色表示
                Image(systemName: "speaker.wave.3")
                    .foregroundColor(volume > 0.7 ? .blue : .gray)
            }
            .padding(.horizontal) // 左右に余白
            .font(.title) // アイコンのサイズ
        }
        .padding() // 全体に余白を追加
    }
}

// プレビューでアプリの見た目を確認
#Preview {
    ContentView()
}
```

SliderはSwiftUIで数値をスライドして入力するためのビューです。ユーザーが直感的に数値を調整できます。

## Sliderの基本構文

```swift
Slider(value: $stateVariable, in: 範囲, step: ステップ)
```
// スライダーの値を@State変数（$記号付き）とバインドし、範囲や刻み幅を指定します。

- **valueパラメータ**: @State変数とのバインディング（$記号必須）
- **inパラメータ**: スライダーの値の範囲（例: 0...1, 0...100）
- **stepパラメータ**: スライダーの刻み幅（例: 0.1, 1.0）

## @Stateとのバインディング

```swift
@State private var volume = 0.5
```
// スライダーで操作する値を保存する変数です。

```swift
Slider(value: $volume, in: 0...1, step: 0.1)
```
// $volumeで状態変数とバインディングし、スライダー操作で値が変化します。

SliderはFloatまたはDouble型の@State変数とバインドします。スライダーの移動に合わせて値がリアルタイムに更新されます。

## 範囲とステップの指定

```swift
// 0.0から1.0まで、0.1刻み
Slider(value: $value, in: 0...1, step: 0.1)
```
// 0から1まで0.1刻みで値を変更できます。

```swift
// 0から100まで、5刻み
Slider(value: $brightness, in: 0...100, step: 5)
```
// 0から100まで5刻みで値を変更できます。

```swift
// -50から50まで、1刻み
Slider(value: $temperature, in: -50...50, step: 1)
```
// -50から50まで1刻みで値を変更できます。

## 数値の変換と表示

```swift
@State private var volume = 0.5
```
// スライダーの値を保存する変数です。

```swift
Text("音量: \(Int(volume * 100))%") // 0.5 → 50%
```
// volumeの値をパーセント表示に変換しています。

```swift
Text("温度: \(temperature)°C")     // そのまま表示
```
// temperatureの値をそのまま表示しています。

## 条件分岐による動的UI

```swift
// 値に応じて色が変わる
Image(systemName: "speaker.wave.1")
    .foregroundColor(volume < 0.3 ? .gray : .blue)
```
// volumeが0.3未満なら灰色、0.3以上なら青色で表示します。

```swift
// 値に応じてアイコンが変わる
Image(systemName: volume > 0.7 ? "speaker.wave.3" : "speaker.wave.1")
```
// volumeが0.7より大きい場合は大きいスピーカーアイコン、それ以外は小さいアイコンを表示します。

## スタイルのカスタマイズ

```swift
Slider(value: $value, in: 0...1)
    .accentColor(.blue)        // スライダーの色
    .padding(.horizontal)      // 周囲の余白
```
// スライダーの色や余白を変更できます。

## 注意点

1. **数値型**: FloatまたはDouble型の@State変数を使用します
2. **範囲指定**: inパラメータで必ず範囲を指定します
3. **ステップ**: stepパラメータは省略可能（連続値になります）
4. **$記号**: バインディングには必ず$記号が必要です
5. **表示変換**: 必要に応じてInt()で整数に変換して表示します
6. **動的UI**: 値に応じて色やアイコンを変更できます

Sliderは音量調整、明るさ設定、温度調整など、数値入力が必要な場面で活躍します。
