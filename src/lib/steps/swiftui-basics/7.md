## カウンターアプリ

カウンターアプリでは、SwiftUIの基本的な概念をすべて活用した実践的な例となっています。

<img src="/images/swiftui-basics/7_1.png" alt="カウンターアプリ" class="mobile-screenshot" />

```swift title="ContentView.swift"
import SwiftUI
struct ContentView: View {
    // カウントの値を保存する状態変数
    @State private var count = 0
    
    // 画面に表示する内容を定義
    var body: some View {
        // 縦方向に要素を並べるコンテナ
        VStack(spacing: 30) {
            // 現在のカウント数を表示するテキスト
            Text("\(count)")
                .font(.system(size: 80, weight: .bold)) // 大きな太字フォント
                .foregroundColor(countColor()) // カウントに応じた色
            
            // 横方向にボタンを並べるコンテナ
            HStack(spacing: 40) {
                // マイナスボタン
                Button(action: {
                        // カウントを1減らす                    
                        count -= 1
                }) {
                    // マイナスアイコンを表示
                    Image(systemName: "minus.circle.fill")
                        .font(.system(size: 60)) // アイコンのサイズ
                        .foregroundColor(.red) // 赤色
                }
                
                // プラスボタン
                Button(action: {
                    // カウントを1増やす
                    count += 1
                }) {
                    // プラスアイコンを表示
                    Image(systemName: "plus.circle.fill")
                        .font(.system(size: 60)) // アイコンのサイズ
                        .foregroundColor(.green) // 緑色
                }
            }
            
            // リセットボタン
            Button("リセット") {
                // カウントを0に戻す
                count = 0
            }
            .padding() // 内側に余白を追加
            .background(Color.blue) // 青い背景
            .foregroundColor(.white) // 白い文字
            .cornerRadius(10) // 角を丸くする
        }
    }
    
    // カウントの値に応じて色を決める関数
    private func countColor() -> Color {
        if count > 0 {
            return .green // 正の値は緑
        } else if count < 0 {
            return .red // 負の値は赤
        } else {
            return .gray // 0は灰色
        }
    }
}

// プレビューでアプリの見た目を確認
#Preview {
    ContentView()
}
```

## カウンターアプリの概要

このカウンターアプリは、SwiftUIの基本的な概念をすべて活用した実践的な例となっています。カウントの増減とリセット機能を備えたシンプルなインターフェースを通じて、状態管理、レイアウト、ユーザー操作、動的UI変更などの重要な概念を学習できます。

## @Stateによる状態管理

```swift
@State private var count = 0
```

まず、`@State`プロパティラッパーを使って`count`という状態変数を宣言しています。この変数は`private`で宣言されており、ビュー内部でのみ変更可能になっています。`count`の初期値は0です。

## UIレイアウトの構成

```swift
VStack(spacing: 30) {
    Text("\(count)")
        .font(.system(size: 80, weight: .bold))
        .foregroundColor(countColor())
    
    HStack(spacing: 40) {
        // プラスとマイナスのボタン
    }
    
    // リセットボタン
}
```

UIのレイアウトには`VStack`（垂直スタック）を使用し、`spacing: 30`で各要素間に30ポイントの間隔を設けています。一番上には現在のカウント値を表示する`Text`ビューがあり、`font(.system(size: 80, weight: .bold))`で非常に大きな太字フォントを指定し、`foregroundColor(countColor())`で動的に色を変更しています。

## ボタンの機能実装

中央には`HStack`（水平スタック）でプラスとマイナスのボタンを配置し、`spacing: 40`でボタン間に40ポイントの間隔を設けています。

**マイナスボタン**
```swift
Button(action: {
        count -= 1
}) {
    Image(systemName: "minus.circle.fill")
        .font(.system(size: 60))
        .foregroundColor(.red)
}
```

**プラスボタン**
```swift
Button(action: {
    count += 1
}) {
    Image(systemName: "plus.circle.fill")
        .font(.system(size: 60))
        .foregroundColor(.green)
}
```

プラスボタンは無条件でcountをインクリメントします。どちらのボタンも`Image(systemName:)`を使ってシステムアイコンを表示し、適切な色とサイズを設定しています。

**リセットボタン**
```swift
Button("リセット") {
    count = 0
}
.padding()
.background(Color.blue)
.foregroundColor(.white)
.cornerRadius(10)
```

一番下には「リセット」ボタンを配置し、`count`を0に戻す機能を持っています。このボタンは`padding()`、`background(Color.blue)`、`foregroundColor(.white)`、`cornerRadius(10)`で青い背景の角丸ボタンとしてスタイル化されています。

## 動的色の変更機能

```swift
private func countColor() -> Color {
    if count > 0 {
        return .green
    } else if count < 0 {
        return .red
    } else {
        return .gray
    }
}
```

最後に、`countColor()`というプライベート関数を定義しています。この関数は`count`の値に応じて`Color`を返却します。countが正の値なら緑、負の値なら赤、0なら灰色を返します。これにより、カウントの状態が視覚的にわかりやすくなっています。
