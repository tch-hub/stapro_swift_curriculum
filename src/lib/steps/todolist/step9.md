## 1. 初期データの定義

`SwiftData/Constants.swift` を開き、タブとそれに紐づくタスクのリストを定義します。
以下のコードを記述してください。

```swift title="SwiftData/Constants.swift"
import Foundation

// 初期データの定義：タブ名と、それに紐づくタスク名のリスト
// ("タブ名", ["タスク1", "タスク2", ...]) という形式で記述
let INITIAL_TODO_TABS = [
    ("リスト", [
        "アイテムA",
        "アイテムB",
        "アイテムC"
    ]),
    ("宿題", [
        "作文",
        "テスト勉強",
        "レポート"
    ]),
    ("買い物", [
        "食料品",
        "日用品",
        "衣類"
    ])
]
```

---

このステップでは「裏側の処理(初期データの定義)」を作りました。画面には何も変化が現れませんが、次のステップ以降でこのデータを使ってアプリを初期化していきます。

## 練習問題は必要ありません

このステップは**データ構造の定義**に特化しているため、練習問題を用意していません。理由は以下の通りです。

- **実行環境での動作確認ができない**: 定数の定義だけでは、アプリの画面に何も表示されないため、学習者が自分の実装が正しいか判断できません。

- **次のステップで実装を通じて検証できる**: step10以降で、このデータを実際に使用するコードを書く際に、定数の定義が正しく機能しているか確認できます。

- **概念よりも「次への準備」**: 定数定義はシンプルな構文学習よりも、「次のステップで使うためのセットアップ」という役割が大きいです。

したがって、このステップで「タプルの配列による複合データの構造化」という概念を理解した上で、次のステップで実際にこのデータを処理していく過程で、深い理解が得られます。
