# Expansion

Expansion contains creation flows and allows lightweight editing of an element.

## Element

```html
<details>
  <summary>...</summary>
  <...>...</...>
</details>
```

## Example multiple levels

```html
<details>
  <summary>...</summary>
  <details>
    <summary>...</summary>
    <details>
      <summary>...</summary>
      <...>...</...>
    </details>
  </details>
</details>
```

## Example custom summary

```html
<details>
  <summary class="none">
    <...>...</...>
  </summary>
  <...>...</...>
</details>
```
